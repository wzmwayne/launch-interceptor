# 文件传输应用

一个支持HTTPS加密、设备间直接传输和实时进度条的文件传输应用。

## 新功能

### 1. HTTPS加密传输
- 应用尝试使用HTTPS加密传输
- 如果系统不支持OpenSSL，会自动回退到HTTP
- 提供更安全的文件传输体验

### 2. 设备间直接传输
- 自动发现局域网内的其他设备
- 支持设备间直接文件传输
- 实时显示在线设备列表

### 3. 上传进度条
- 实时显示文件上传进度
- 支持多文件同时上传
- 优雅的进度条动画效果

## 使用方法

### 启动应用
```bash
./start.sh
```
或
```bash
python app.py
```

### 访问应用
- 本地访问: http://127.0.0.1:5000
- 局域网访问: http://[你的IP]:5000

### 功能说明

#### 文件上传
1. 拖拽文件到上传区域
2. 或点击"选择文件"按钮
3. 查看实时上传进度

#### 设备间传输
1. 应用会自动发现局域网内的其他设备
2. 在设备列表中选择目标设备
3. 点击"传输文件"按钮
4. 等待对方接受传输请求

#### 接收文件
1. 当有设备请求传输文件时，会弹出确认对话框
2. 点击"接受"接收文件
3. 或点击"拒绝"取消传输

## 技术特点

- **暗色主题**: 现代化的安卓风格UI设计
- **WebSocket**: 实时通信和进度更新
- **响应式设计**: 适配各种设备屏幕
- **安全性**: 支持HTTPS加密传输

## 系统要求

- Python 3.7+
- Flask
- Flask-SocketIO
- qrcode
- Pillow

## 安装依赖

```bash
pip install -r requirements.txt
```

## 注意事项

- 如果系统不支持OpenSSL，应用会回退到HTTP模式
- 设备间传输需要设备在同一局域网内
- 大文件传输可能需要较长时间，请耐心等待
